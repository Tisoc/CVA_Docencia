Barra de estado

Muestra información adicional sobre los elementos que tenemos seleccionados. Esta barra es opcional, para activarla, vaya al menú Ver (View), y haga clic en Barra de estado (Status Bar).

Windows XP

A partir de Windows Vista se modificó la barra de estado y se convierte en panel de detalle, donde se muestra la información sobre el archivo seleccionado.

Para activarla en Windows 10, vaya al menú Vista y seleccione Panel de detalles

Windows XP

En Windows Vista y 7 se encuentra activada de manera predeterminada, para desactivarla, vaya al menu Organizar (Organize) y haga clic en Diseño (Design) y Panel de Detalles (Status Bar).
